About Me

Halo! Saya Ahmad Hibrizi, seorang desainer visual dengan fokus pada desain grafis dan UI/UX. Saya berkomitmen untuk menciptakan desain yang fungsional, menarik, dan mudah digunakan. Saya selalu berusaha untuk menggabungkan kreativitas dengan kebutuhan pengguna dalam setiap proyek yang saya kerjakan.


Saya memiliki pengalaman dalam membuat berbagai poster promosi untuk produk makanan, minuman, dan barang, serta merancang desain antarmuka pengguna (UI) untuk website nonton anime menggunakan Figma. Selain itu, saya pernah menjadi editor konten selama dua bulan untuk tugas kelompok sekolah, membantu guru dalam mendokumentasikan dan mengedit tugas kuliahnya, serta berkontribusi dalam pembuatan logo dan desain pakaian dinas harian (PDH) untuk kunjungan industri sekolah. Saya juga menjalani Praktik Kerja Lapangan (PKL) selama lima bulan di PT Lambda Sinergi Telekomunikasi sebagai graphic designer, dengan tugas utama mendesain konten visual untuk kebutuhan promosi dan branding perusahaan.
